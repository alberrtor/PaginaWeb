PaginaWeb
=========

Aplicación para dar de alta los empleados del plantel, principalmete llevar el registro de los cumpleaños.

Herramientas utilizadas:
Jsp, servlet, extjs, ajax, json, tomcat, jdbc.

Notas importantes:
Se utiliza la base de datos del plantel para la autentificación y la tabla "usuario" de la base de datos "pagina_web" 
para el registro de empleados.
